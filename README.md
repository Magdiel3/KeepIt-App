# KeepIt-App
Main app for the user of KeepIt

## Installation
Clone project with
```
git clone https://github.com/Magdiel3/KeepIt_App.git
cd KeepIt_App/
```
And run the following commands to start local server
```
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
python app.py
```
